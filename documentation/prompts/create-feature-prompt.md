# Prompt de criação de feature

Primeira, entenda o que já existe na aplicação referente a feature que a ser criada
Entenda o PRD do projeto (documentation/prd.md)
Entenda a arquitetura da aplicação (documentation/architecture.md)
Depois de implementado a feature, execute o typecheck usando o ``npm run typecheck``
Depois crie os testes unitários dos services envolvendo a feature e execute com ``npm run test``
Retorne um resumo do que foi feito da implementação
Se precisar, atualize o PRD ou o documento de arquitetura com o que foi feito
